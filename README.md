# Simple-SAS-SelfRating
intact name is Self-Rating Anxiety Scale.

## 用途
很简单，这是一个用来检测是否有焦虑症的小程序，认真的回忆一周内是否有问题中的情况，根据程度选择后面的选项，最终能够得出焦虑症的程度或者是否具有焦虑症。
## 运行环境
仅在 PHP 5.4 下测试过。
